## 📌 Project Overview
This project provides a comprehensive analysis of the **2024 Indian General Election** results. By integrating multiple datasets covering constituency-level details, party performance, and state-wise outcomes, this dashboard offers a deep dive into the democratic landscape.

## ❓ Problem Statement
Analyzing election data in its raw form is challenging due to the massive volume of candidates. This dashboard provides an interactive interface to:
* **Identify** winning trends and party dominance across states.
* **Compare** EVM vs. Postal vote distributions.
* **Analyze** victory margins to identify "hot seats."
* **Track** individual candidate performance.

## 📂 Dataset Information
The project uses five core datasets:
1. `constituencywise_details.csv`: Candidate-level vote counts.
2. `constituencywise_results.csv`: Summary of winners per seat.
3. `partywise_results.csv`: Total seats won by each party.
4. `states.csv`: State ID mapping.
5. `statewise_results.csv`: Comparative analysis of leads and trails.
